Keeptrack is an open source application that let you save and review everything you read, watch, listen or play.

## Hosting

Applications are Cloud Native by design, they can be deployed to any kind of infrastructure.

A SaaS free version is currently available for early adopters (contact the repository owner to get an access).

## Software design

3 tier application:

- Frontend: Blazor Server application (.NET 10)
- Backend: ASP.NET Web API application (.NET 10)
- Database: MongoDB

For more details, see [CONTRIBUTING.md](CONTRIBUTING.md) - any help is more than welcome 🙂
